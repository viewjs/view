# View

## Installation

browser:

```bash
component install view
```

## Example

```js
var project = require('view');
```

## API

Example API methods.

### .on(event, fn, [capture])

Short description of some method:

```js
project.on('event', function(e){

});
```

### .on(event, selector, fn, [capture])

Another description:

```js
project.on('event', 'a.remove', function(e){

});
```
